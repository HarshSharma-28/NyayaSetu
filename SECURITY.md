# Security Policy

## Supported Versions

We currently support the following versions of NyayaSetu with security updates:

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

Security is a top priority for NyayaSetu. If you discover a vulnerability or a security issue, please do not disclose it publicly.

Instead, please report it immediately by contacting the repository owner directly.
All security vulnerabilities will be promptly investigated and patched in the next release cycle. 

Thank you for helping keep the platform secure.
